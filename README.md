# Brown Sugar Boba portfolio

The source for [brownsugarboba.com](https://brownsugarboba.com), built with React and Vite.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The static site is written to `dist/`. The `public/CNAME` file configures the production custom domain for GitHub Pages.

## Keeping the portfolio current

Product names, descriptions, launch states, and destination URLs live in the `products` array near the top of `src/App.jsx`. The copyright year updates automatically.
